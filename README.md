# MicroEconomy The Game

This is a game designed for Scouts to play during a scout meeting to try and teach them about economics, the stock market and basic business studies.

## Setup

Needs:
- Control Laptop
- Travel Router
- Team Devices (iPads, laptops, phones)

Control Laptop:
Is for the game master, runs containers with:
1. The game script
2. Database
3. Webserver

Travel Router:
This establishes the game Wifi Signal and ensures that all of the team devices can access the local webserver.


## Game Play

The aim of the game is to play the markets to get the most valuable portfolio!

Each round, every team will make a number of trades (buying & selling) to play the odds. They will each also pick a chance event card.

Teams will discuss the best trades, based on information about the markets and gambles on what might happen.

Each team can only see their own chance card until they have made their trades.

## Calculations

1. Trades
2. Government Policy
